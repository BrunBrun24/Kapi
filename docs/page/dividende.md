# 🎯 Objectif de la page « Dividendes »

La page **Dividendes** permet à l'utilisateur de suivre, enregistrer et analyser les revenus générés par les actions détenues dans ses portefeuilles. Elle offre une vision claire des dividendes perçus, par action, par date ou par portefeuille.

---

## 🧭 Parcours utilisateur

### 1. Sélection d’un portefeuille  
L’utilisateur commence par sélectionner un portefeuille existant (préalablement créé sur la page Portefeuilles).

### 2. Affichage des dividendes liés au portefeuille  
Une fois un portefeuille sélectionné, l’utilisateur peut consulter les dividendes associés aux actions présentes dans ce portefeuille. Ces données peuvent être triées et filtrées.

### 3. Ajout manuel d’un dividende  
L’utilisateur peut enregistrer un dividende perçu pour une action donnée (qui doit appartenir au portefeuille sélectionné). Cela comprend :
- le **ticker** concerné,
- le **montant** reçu,
- la **date de versement**,
- éventuellement la **devise** ou d’autres métadonnées.

---

## 👁️ Visualisation des données

L’utilisateur dispose d’un tableau ou d’une liste qui affiche :
- la date de versement,
- le ticker,
- le montant reçu,
- la devise (le cas échéant),
- le total mensuel ou annuel (si agrégation possible).

---

## 🧩 Comportement dynamique

- Le champ **ticker** est limité aux actions présentes dans le portefeuille sélectionné.
- Les dividendes peuvent être filtrés par :
  - date,
  - ticker,
  - montant.
- Possibilité de tri par date ou montant décroissant.
