# 🎯 Objectif de la page « Dashboard »

La page **Dashboard** constitue le point central de visualisation de l’état global des portefeuilles de l’utilisateur. Elle présente de manière synthétique et graphique les informations clés sur les performances, la répartition des actifs et les revenus.

---

## 🧭 Parcours utilisateur

### 1. Sélection d’un portefeuille  
L’utilisateur peut choisir un portefeuille pour afficher ses données spécifiques, ou visualiser un agrégat de tous les portefeuilles.

### 2. Vue d’ensemble financière  
Le tableau de bord affiche :
- la valeur totale du portefeuille,
- les plus-values latentes (gains/pertes),
- les dividendes reçus,
- les performances par action ou secteur.

---

## 👁️ Visualisation des données

Les composants suivants peuvent être présents :
- graphiques de performance (lignes, barres, secteurs),
- répartition géographique ou sectorielle,
- évolution de la valeur dans le temps,
- top performers / worst performers.

---

## 🧩 Comportement dynamique

- Les données changent dynamiquement selon le portefeuille sélectionné.
- Les graphiques sont interactifs (tooltips, filtres).
- Agrégation automatique des données par période, secteur ou devise.
